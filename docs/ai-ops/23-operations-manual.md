# 店主操作手冊

- 更新日：2026-07-18
- 適用對象：店主／後台操作人員
- 本次差異盤點：客戶管理、客戶詳情、技能地圖、每月毛利、銀行參考匯率五項皆已有操作說明，**無缺頁**；已將多銀行比價現況同步進第 7 節。

## 最短流程

1. 到「設定」填店鋪進貨匯率與可用物流方式。
2. 新增商品，需要看毛利時一併填成本與交通路線。
3. 複製商品分享連結給客人。
4. 到「訂單」確認新單。
5. 撿貨、包貨、出貨並更新狀態。
6. 展開訂單查看已定格毛利，或到首頁看每月毛利。

## 1. 老闆首頁與每月毛利

1. 登入後先看首頁的「老闆今日重點」。
2. 「今日／本週訂單」是筆數；「已定格毛利小計」只加總已有金額快照的訂單。
3. 「待確認訂單」是訂單流程狀態；「毛利待補拍」與「尚無快照」是成本資料狀態，三者意思不同。
4. 點「處理訂單」可逐筆處理。
5. 點「查看每月毛利報表」後選月份，可看該月訂單數、已定格毛利、待確認與尚無快照數。

> 截圖留白 1：首頁六張重點卡與「查看每月毛利報表」。

## 2. 上架與分享商品

1. 點下方「商品」，再點「新增商品」。
2. 填商品名稱、一般售價、庫存與規格。
3. 需要看成本毛利時，同時填商品日圓成本並確認店鋪進貨匯率。
4. VIP／批發／夥伴價可以留空；留空會使用一般售價，不會當成 0 元。
5. 商品免攤交通時打開免攤；否則選擇已有成本資料的路線。
6. 儲存後確認商品已啟用。預估毛利顯示「待確認」時，先補齊成本資料。
7. 在商品卡點「複製分享連結」，把連結貼給客人；系統不會自動發訊息。

## 3. 接單與確認

1. 客人從分享頁填資料下單，不需店主代操作。
2. 點下方「訂單」，展開新單核對商品、數量、金額與取貨方式。
3. 依實際進度更新：待確認 → 待付款 → 備貨中 → 已出貨 → 已完成。
4. 成本快照顯示「待確認」時，補齊商品成本、店鋪匯率與路線後，可按一次「補拍」。
5. 「尚無快照」通常是舊單，也可補拍一次。補拍後會永久定格並留下時間。

> 截圖留白 4：新單、狀態選單與補拍按鈕。

## 4. 撿貨、包貨與出貨

1. 到訂單頁下方的撿貨／出貨工具。
2. 開啟撿貨清單或下載撿貨檔，依商品與規格分揀。
3. 包裝前再次核對客人、品項、數量與取貨方式。
4. 實際交寄後才把出貨狀態改為「已出貨」。
5. 客人取貨／交易結束後再改為完成。

> 截圖留白 5：撿貨清單、出貨工具與物流狀態。

## 5. 查毛利

1. 在訂單列表展開該單，找到「成本與單件毛利」。
2. 「金額」表示建單或補拍時已定格；以後改成本或匯率不會改舊單。
3. 「免攤」表示交通成本以 0 定格，而且畫面會明確標示免攤。
4. 「待確認」表示當時成本資料不足，不能當成 0。
5. 「尚無快照」可在資料完整後補拍一次。

> 截圖留白 6：金額、待確認、免攤、尚無快照四種狀態。

## 6. 行程與交通成本

1. 到「設定 → 行程與路線管理」。
2. 先建立行程；行程匯率可以留空，但相關商品成本會顯示待確認。
3. 在行程內新增路線，填預估件數、電車、油資、停車、ETC、紙箱、日本境內運費與包裹數。
4. ETC 必須自己填；沒有費用時填 0，系統不會自動用件數代填。
5. 商品沒有免攤時，在商品表單選擇對應路線。

> 截圖留白 7：行程與路線中的 ETC、日本境內運費、預估件數。

## 7. 店鋪匯率與銀行參考匯率

1. 到「設定 → 店家設定」填店鋪進貨匯率。
2. 到「設定 → 銀行匯率參考」查看已核可銀行的日圓即期賣出價、來源與掛牌時間。
3. 抓不到資料時會顯示「暫時不可用」，不會用 0 或舊值冒充。
4. 點某家銀行的「套用」只會把參考值填進輸入框，仍要自己確認並按儲存。
5. 參考匯率不會自動覆蓋店鋪匯率或行程匯率。

> 截圖留白 8：多銀行參考表、掛牌時間與「套用」。

## 8. 客戶管理與客戶詳情

1. 先到「設定 → 技能地圖」開啟「客戶管理」。
2. 回到「設定 → 客戶管理」，建立客戶代號、姓名、可選手機、等級與常用門市。
3. 客戶列表預設遮罩；需要時點「顯示完整」，系統會留下揭露紀錄。
4. 點客戶卡右側「詳情」，可看基本資料、常用門市與該客戶的訂單歷史。
5. 詳情頁顯示的是訂單定格毛利；待確認不會顯示成 0。
6. 後台新增訂單時可用代號或姓名搜尋客戶；選中後會帶入常用門市。
7. 如果在訂單內手動改選門市，手選結果優先。

## 9. 技能地圖

1. 到「設定 → 技能地圖」查看各種技能與套餐。
2. 技能卡會說明省什麼工、前置條件、風險與開啟後的效果。
3. 前置條件不足時按鈕不能按，卡片會顯示原因；先補齊條件再回來。
4. 單項技能可以開關；套餐會先顯示差異，再由你確認套用。
5. 高風險技能會多一次確認。
6. 關閉日常技能後，對應入口會隱藏；技能地圖本身仍會保留，方便重新開啟。
7. 首登推薦問卷目前尚未開放，請直接人工選擇套餐。

### FAQ：為什麼客戶管理或成本欄位看不到？

這些是進階功能，尚未開啟時，日常入口會先隱藏。資料不會因此被刪除。

1. 到底部「設定」，點「技能地圖」。
2. 想使用客戶管理：找到「客戶管理」，先確認卡片顯示「可開啟」，再按「開啟」。
3. 想使用成本欄位：在頁面上方「建議套餐」選擇「成本套餐」，按「預覽套餐」。
4. 在「套餐差異預覽」確認這次可開啟的項目，再按「套用低風險項目」。
5. 若技能卡顯示「缺前置」，請先照「開啟前要準備」補資料，再回來開啟。
6. 高風險項目會出現第二次確認；勾選「我已閱讀影響與風險，確認要開啟」，再按「確認開啟」。
7. 開啟完成後回到「設定」；「客戶管理」或相關成本入口會立即出現，不需要重新整理。

若按鈕仍不能按，先看技能卡的「缺前置」原因；不要反覆按，也不要把缺資料當成 0。

## 10. 匯出客戶與訂單

1. 客戶與訂單匯出預設為遮罩版。
2. 匯出前會顯示筆數並要求確認。
3. 明文版要再勾一次確認；下載後請妥善保管並在使用後刪除。
4. 訂單毛利待確認時，檔案會寫「待確認」，不會寫成 0。
5. 匯出與顯示明文都會留下操作紀錄，紀錄不保存明文個資。

> 截圖留白 12：遮罩版／明文版的二次確認。

## 11. 付款末五碼人工對帳

1. 客人可在待確認／待付款期間填付款末五碼；這只是對帳線索，不會自動改成已付款。
2. 後台可用末五碼、金額與狀態篩選，再由店主核對實際入帳。
3. 末五碼不是完整帳號，也不是付款證明。

## 操作提醒

- 看到「待確認」不要自行當成 0 記帳。
- 訂單編輯不等於已通知客人；訊息模板也只會複製，不會自動發送。
- 取消訂單會保留紀錄；有物流或完成紀錄的訂單不要隨意刪除。
- 首登問卷、賣貨便 XLSM 匯出與購物金目前都還不是可操作功能。
