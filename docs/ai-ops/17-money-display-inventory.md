## 違規／待修清單

1. **首頁最近訂單金額幣別不清楚**：`Dashboard.tsx:329` 使用 `$`，不是全站慣例 `NT$`；而且顯示來源是 `totalPrice`（商品總額），沒有文字說明是否含運費。建議改成 `NT$` 並標「商品小計」或改用明確的 `orderTotal`。
2. **部分交易預覽仍以 JavaScript Number 做顯示運算**：公開下單、購物車、後台新增／編輯訂單預覽會用 Number 相乘或相加（例如 `PublicOrder.tsx:151-153`、`PublicCart.tsx:381-383`、`EditOrderDialog.tsx:295-317`）。正式寫入與成本毛利鏈已有 ExactDecimal 防線，但預覽若遇小數價仍可能與伺服器定格值有顯示差異。建議另包把「預覽用」運算也收斂到共用 decimal 顯示 helper；本盤點不改碼。

## 結論

成本與毛利顯示已遵守「預估／定格／待確認／免攤」標記，沒有把缺值靜默顯示成 0。主要缺口是首頁一處使用不明確的 `$`，以及交易預覽仍有 Number 顯示運算；兩者均未在本盤點包修改。
